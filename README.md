# AI Chatbot Mentor

A module-specific AI chatbot mentor designed to provide focused learning guidance instead of generic responses.

## Overview
This project allows users to select a learning module (such as Python, SQL, or Machine Learning), and the chatbot responds strictly within the chosen domain. This helps reduce confusion and encourages structured learning.

## Features
- Module-based mentoring system
- Domain-restricted responses using prompt engineering
- Interactive UI built with Streamlit
- Beginner-friendly guidance

## Tech Stack
- Python
- Streamlit
- LangChain
- Large Language Models (LLMs)

## How to Run
```bash
pip install -r requirements.txt
streamlit run main.py
